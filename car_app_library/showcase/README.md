# Smart In-Vehicle Connectivity Assistant

This project is based on the Android for Cars showcase sample and reshaped into a
connectivity-focused infotainment demo for portfolio and interview use.

## What it demonstrates

- Vehicle connectivity monitoring for Bluetooth, Wi-Fi, USB, and phone projection
- Infotainment media state presentation for a simulated in-vehicle environment
- Event logging for issue analysis and system behavior tracing
- Reconnect flow summaries for recovery and timeout scenarios

## Main screens

- Home overview with scenario switching
- Connection dashboard
- Media center
- Event log
- Reconnect flow

## Why it fits Garmin Connectivity Team

The project highlights Android for Cars development, system state handling, multimedia
interaction, and debugging-oriented event tracking, all of which map well to automotive
connectivity and infotainment work.
